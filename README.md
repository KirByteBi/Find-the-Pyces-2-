## 📖 **Sobre el Proyecto**

Web bilingüe (ES/EN) que documenta el universo de Find the Pyces 2, incluyendo:

- Sistema de login con registro (localStorage)
- Easter eggs secretos
- Información sobre Bitlands, Pyces, Portalogía y dificultades
- Enlaces a la comunidad (Roblox, Discord)
- Diseño responsive con colores azul, lila y rosa
